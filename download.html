常见安装问题与解决方案

Windows 安装时提示"此应用无法在你的电脑上运行"？
请确认下载正确架构版本。32 位系统选 x86 版本，64 位系统选 x64 版本。在"设置 → 系统 → 关于"中查看系统类型。也可尝试右键以管理员身份运行安装程序。
Android 提示"安装被阻止"或限制未知来源？
进入"设置 → 安全"找到"未知来源"选项并开启。不同品牌路径略有差异：华为在"设置 → 安全和隐私 → 更多安全设置"；小米在"设置 → 隐私保护 → 特殊权限"中设置。
iOS 搜索不到 Telegram 或无法下载？
请确认 Apple ID 地区设置为非中国大陆地区。部分区域 App Store 不显示此应用，需使用对应地区的 Apple ID 登录后重新搜索下载。
macOS 提示"无法验证开发者"无法打开？
这是 macOS Gatekeeper 安全机制。打开"系统设置 → 隐私与安全性"，页面底部会显示被阻止的应用，点击"仍要打开"即可。从 Mac App Store 下载的版本无需此操作。
收不到短信验证码应该怎么办？
首先确认手机号含国家区号 +86 且输入无误。等待 60 秒后可点击重新发送。如多次收不到可尝试语音验证码。同时检查手机短信拦截软件是否误拦了国际短信。
如何确认下载的是官方正版而非山寨应用？
Windows/Mac 版数字签名应为 Telegram FZ-LLC。Android 建议从 Google Play 或 telegram.org 官方下载。iOS 在 App Store 搜索时核对开发者名称。本站所有下载链接均为官方直链，安全无捆绑。
